Introduction

Fable Python is a workbench for writing python, running scripts and plotting python NumPy arrays.

This application part of a series of Fable graphical applications based on Eclipse.

Fable Python is written in Java and does all plotting using pure java libraries (Jean-Luc Pons JLChart for 1D, SWT's Image for 2D and OpenGL for 3D).

Jepp

Fable Python uses the fabulous Java Embedded Python Jepp (http://jepp.sourceforge.net/) for running Python in Java.

Open Source

Fable is an Open Source project hosted on SourceForge.

The source code for fable.python can be consulted online at :
